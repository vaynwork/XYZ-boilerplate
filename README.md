# XYZ-boilerplate
a boilerplate setup for using XYZ

### Folder Structure

This is the folder strucuture for a XYZ project, designed to keep everything tidy and easy to find

```bash
├── 📂assets
│   ├── 📂img
│   │   ├── 📂favicon
│   ├── 📂css
│   │   ├── XYZ_Theme.css
│   ├── 📂html
│   │   ├── 📂page
│   │       ├── footer.html
│   │       ├── header.html
│   └── 📂js
├── 🌐index.html
├── 📰XYZ_Config.json
├── 📰README.md
├── 📰site.webmanifest
└── .gitignore
```
