### Vuwnu Web Tools Boilerplate
a boilerplate setup for using Vuwnu Web Tools

#### Folder Structure

This is the folder structure for a project, designed to keep everything tidy and easy to find

```bash
├── 📂assets
│   ├── 📂img
│   │   ├── 📂favicon
│   ├── 📂css
│   │   ├── style.css
│   ├── 📂html
│   │   ├── 📂page
│   │       ├── footer.html
│   │       ├── header.html
│   └── 📂js
├── 🌐index.html
├── 📰README.md
├── 📰site.webmanifest
└── .gitignore
```
